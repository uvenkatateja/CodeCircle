## Getting Started

1. Install the extension
2. Click the CodeCircle icon in the Activity Bar
3. Connect with GitHub or continue as guest
4. Start seeing your friends' coding status!

## Privacy

- **Your choice**: Control who sees your status (everyone, followers, following, or invisible)
- **Local chat**: Messages are stored on your device, not our servers
- **Minimal data**: We only store your GitHub ID and username

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `codecircle.visibilityMode` | `everyone` | Who can see your status |
| `codecircle.shareProject` | `true` | Share project name |
| `codecircle.shareLanguage` | `true` | Share programming language |
| `codecircle.shareActivity` | `true` | Share activity status |
| `codecircle.showInStatusBar` | `true` | Show online count in status bar |
| `codecircle.showInExplorer` | `true` | Show friends in Explorer |

## Commands

- `CodeCircle: Connect GitHub` - Sign in with GitHub
- `CodeCircle: Continue as Guest` - Use without GitHub
- `CodeCircle: Create Invite Link` - Generate invite code
- `CodeCircle: Accept Invite Code` - Connect with a friend
- `CodeCircle: Sign Out` - Sign out of CodeCircle
